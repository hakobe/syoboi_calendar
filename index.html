About

Simple gem for SyoboiCalendar to search Japanese anime lineup

Require

Ruby 1.9.2 or higher

Usage

# search_anime.rb
require "syoboi_calender"

client   = SyoboiCalendar::Client.new
programs = client.search(:range => "2012/4/1-2012/4/30")

puts "name    : #{programs[0].name}"
puts "channel : #{programs[0].channel_name}"
puts "start   : #{programs[0].start_time}"

$ gem install syoboi_calendar
$ ruby search_anime.rb
name    : さんかれあ #1 私が…ゾンビに…なったら
channel : 2012-04-06 02:50:00 +0900
start   : TBS
